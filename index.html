Kolmogorov-Arnold Networks (KAN)

Neural Networks Course

Kolmogorov-Arnold Networks (KAN)

Project Information

  • Description: Developed in 2024, Kolmogorov-Arnold Networks (KANs) are a novel approach to deep learning, inspired by the Kolmogorov-Arnold representation theorem. Unlike traditional Multi-Layer Perceptrons (MLPs), where nodes have fixed activation functions, KANs utilize learnable activation functions on the edges (weights). KANs eliminate linear weights and instead replace them with univariate functions parametrized as splines.
  • Date: Jul 2024
  • Keywords: KAN, Neural Networks, MLP Replacement, Learnable Activation Functions, Kolmogorov-Arnold Theorem, Deep Learning
  • Code: GitHub Repository

Project Summary

The project implements KANs with a custom-built neural network structure, offering flexibility in designing layers and nodes. The Optimizer class implements gradient descent, allowing backpropagation through the network to minimize the loss. The implementation includes training the model over a series of epochs, applying the KAN to sample datasets to showcase its effectiveness.

Key Features

  • Learnable Activation Functions: Activation functions are no longer fixed. Instead, they are learnable, dynamic functions applied to weights, offering greater flexibility.
  • No Linear Weights: KANs eliminate linear weights and replace them with splines, which are univariate functions parametrized to adjust based on data and tasks.
  • Accuracy and Efficiency: KANs offer higher accuracy and more efficient scaling than traditional MLPs, making them suitable for mathematical modeling and physics applications.
  • Visualization: Easily visualize the networks computational graph, allowing human users to interact with and interpret KANs operations.

Optimizing Position Sizing for Trading

Project Information

  • Description: This project optimizes the position sizing strategy for trading by using mathematical modeling and visualization. It compares the performance of different position sizes, calculates the optimized position size based on return and risk, and uses the GEKKO library for optimization.
  • Date: Nov 2022
  • Keywords: Position Sizing, Trading, Optimization, GEKKO, Python, Mathematical Modeling
  • Code: GitHub Repository

Project Summary

In this project, I explore the impact of different position sizes on account balance performance using Python, Pandas, and Matplotlib for visualization. I begin by plotting account performance for two position sizes (70% and 10%) and observe that smaller position sizes tend to show more stable growth, while larger positions may lead to greater volatility and potential downtrends.

The key objective is to identify the most efficient position size that maximizes returns while minimizing risks. Using the GEKKO optimization suite, I formulate a mathematical model based on risk and reward (R&R), win rate, and other factors.

Visualization and Performance Analysis

  • Position Sizing Comparison: I compare the performance of position sizes ranging from 10% to 60% using multiple plots, showing how each size affects balance growth.
  • Optimization Results: Using the GEKKO optimization solver, I calculate the optimal position size for maximizing returns based on model assumptions, generating an optimal lambda value.
  • Dynamic Visualization: The relative return per lambda is visualized, allowing users to understand the impact of different position sizes on their capital.
